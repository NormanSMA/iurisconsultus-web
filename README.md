# Iuris Consultus - Web Corporativa

Sitio web corporativo moderno para la firma de abogados y notarios **Iuris Consultus** en Nicaragua. Este proyecto presenta un diseño elegante, profesional y responsivo ("Refined Legal Minimalist") optimizado para una experiencia de usuario premium.

## 🚀 Tecnologías

Este proyecto está construido con:

- **[Astro](https://astro.build/)** - Framework para sitios con contenido optimizado.
- **[Tailwind CSS v4](https://tailwindcss.com/)** - Estilizado moderno y eficiente.
- **[Lucide React](https://lucide.dev/)** - Iconografía corporativa limpia.
- **[Astro React Integration](https://docs.astro.build/en/guides/integrations-guide/react/)** - Para componentes interactivos dinámicos.

## ✨ Características Principales

- **Hero con Video Background**: Escena corporativa en alta definición con efecto de fondo fijo (parallax).
- **Diseño Responsivo**: Menú móvil profesional con panel deslizante y bloqueo de scroll.
- **Blog Integrado**: Sección de últimas publicaciones en el inicio y página de blog completa con filtrado por categorías.
- **Formulario de Contacto Avanzado**: Validación en tiempo real (inline) y feedback visual inmediato.
- **Mapa Satelital**: Integración personalizada de Google Maps con vista satelital sobre Plaza Centroamérica.
- **Animaciones de Scroll**: Revelaciones suaves de elementos mediante Intersection Observer.
